Class NotAnInputNodeError

Raised when tx.set targets a derived node.

Remarks

The runtime universe is two primitives: InputNode<T> (a writable Behavior) and DerivedNode<T> (a Behavior computed from other Behaviors). A derived value's meaning is derived(t) = f(b1(t), …, bn(t)) — a pure function of its dependencies at the same commit moment. Permitting tx.set on a derived would tear that equation, so the engine rejects the call at the API boundary rather than accept a write that contradicts the value's denotational definition.

Param: id

The derived node id that was the target of tx.set.

StaticstackTraceLimit

stackTraceLimit: number

The Error.stackTraceLimit property specifies the number of stack frames collected by a stack trace (whether generated by new Error().stack or Error.captureStackTrace(obj)).

The default value is 10 but may be set to any valid JavaScript number. Changes will affect any stack trace captured after the value has been changed.

If set to a non-number value, or set to a negative number, stack traces will not capture any frames.
