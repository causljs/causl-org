Interface IRInput

IR projection of an input node — a writable Behavior captured at export time.

Remarks

The §4 two-primitive discipline lives here. What earlier drafts called a "resource" is an IRInput whose value carries the application's ResourceState<T> payload — composition at the application layer, not a separate kind on the wire (#359).

interface IRInput {
    graphId: string;

    kind: "input";
    serializable: boolean;
    value: unknown;
}
