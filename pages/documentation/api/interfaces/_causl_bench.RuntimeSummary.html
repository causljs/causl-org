Interface RuntimeSummary

Wall-clock summary for the sampled iterations of a single (library × scenario × scale) cell. All times are milliseconds.

samples is the count of timed iterations behind the summary (warm-up runs are excluded by harness implementations).

interface RuntimeSummary {
    mean: number;
    median: number;
    p95: number;
    samples: number;
